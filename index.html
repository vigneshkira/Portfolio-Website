M Mart (Smart Store Concept)

A proof-of-concept for a checkout-free convenience store, using multi-camera Computer Vision and sensor fusion.

Problem Statement

Long checkout lines are a major pain point in retail. This project explores a "grab-and-go" system similar to Amazon Go.

Key Features

  • ✅ Object detection (YOLOv5) to identify products.
  • ✅ Person tracking across multiple camera feeds.
  • ✅ Sensor fusion (weight sensors + cameras) for accuracy.
  • ✅ A virtual cart system that updates in real-time.

Tech Stack

Software: Python, OpenCV, TensorFlow, ROS (for message passing), MQTT
Hardware: Jetson Nano, IP Cameras, RFID, Weight Sensors

My Role

AI & Robotics Engineer. My focus was on developing the multi-camera tracking algorithm and integrating the weight sensor data with the visual data for a robust "virtual cart" system.
